# nr-v2x-west-to-east-highway

Исследовательский NR sidelink highway-сценарий с KPI в SQLite.

- Исходник: `src/nr/examples/nr-v2x-examples/nr-v2x-west-to-east-highway.cc`
- Ключевые строки: `:25`, `:435`, `:507`

## Запуск

```bash
scenarios/nr-v2x-west-to-east-highway/run.sh
```

## Что сохраняется

- Лог: `analysis/scenario_runs/<date>/nr-v2x-west-to-east-highway.log`
- SQLite: `analysis/scenario_runs/<date>/artifacts/<simTag>-nr-v2x-west-to-east-highway.db`
- Графики: `analysis/scenario_runs/<date>/figures/nr-v2x-west-to-east-highway/*.png`
